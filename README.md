# 濮阳市天气预报网站
纯静态前端天气站点，基于Open-Meteo免费开源气象接口，无需API密钥，GitHub Pages一键部署
覆盖：华龙区、濮阳县、清丰、南乐、范县、台前

## 功能清单
1. 实时温度、体感温度、湿度、风向风速
2. 未来7天天气预报
3. 濮阳六区县一键切换查看天气
4. 移动端自适应
5. 永久免费无调用次数限制，无需注册

## 部署教程
1. GitHub新建仓库，上传全部代码
2. Settings -> Pages，main分支根目录部署
3. 等待1分钟生成在线网址

## 技术栈
HTML5 + Bootstrap5 + 原生JS + Open-Meteo 免费气象API
开源协议：MIT
数据来源：Open-Meteo 开源气象项目
