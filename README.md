# 濮阳市天气预报网站
纯静态前端天气站点，基于免费公开天气API，GitHub Pages一键部署
专属濮阳本地天气：华龙区、濮阳县、清丰、南乐、范县、台前统一查询

## 功能清单
1. 实时温度、体感温度、湿度、风向风力
2. 今日24小时逐时预报
3. 未来7天天气预报
4. 全市空气质量AQI、污染等级
5. 生活出行指数：穿衣、洗车、紫外线、运动、感冒
6. 濮阳六区县切换查看天气
7. 移动端自适应

## 部署教程
1. GitHub新建仓库，上传全部代码
2. Settings -> Pages，main分支根目录部署
3. 等待1分钟生成在线网址

## 技术栈
HTML5 + Bootstrap5 + 原生JS + 和风天气免费API
